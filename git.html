Git

Git — это распределенная система контроля версий, разработанная Линусом Торвальдсом.

Видео про Git

Основные особенности

  • Распределенная архитектура
  • Мощная система ветвления и слияния
  • Высокая производительность

Основные команды Git

  • git init — создать новый репозиторий
  • git clone [URL] — клонировать существующий репозиторий
  • git commit -m "Сообщение" — сохранить изменения

Пример использования


$ git init
$ git add .
$ git commit -m "Первый коммит"
$ git push origin main
