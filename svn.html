Subversion (SVN)

Apache Subversion (SVN) — это централизованная система контроля версий, популярная в корпоративной среде.

Основные особенности

  • Централизованное хранение кода
  • Поддержка истории изменений и резервных копий
  • Гранулированное управление доступом

Основные команды SVN

  • svn checkout [URL] — клонировать репозиторий
  • svn commit -m "Сообщение" — отправить изменения
  • svn update — обновить локальную копию

Пример использования


$ svn checkout http://example.com/repo
$ cd repo
$ svn add .
$ svn commit -m "Первый коммит"
